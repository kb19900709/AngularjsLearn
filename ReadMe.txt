Angularjs

1.	two way data binding 特性，省去很多樣板式代碼。 如擷取表單最新值
2.	宣告式語法，讓開發人員更迅速的了解 html 的狀態與動作
3.	關注點分離結構(MVC or MVVM)，更迅速的針對某個節點開發與維護
4.	注入式的語法，讓寫過的服務與商業邏輯(service)可重用
5.	豐富的預設指令集(directives)，以及可自訂的指令。增加 DOM 的狀態與動作
6.	易於測試
